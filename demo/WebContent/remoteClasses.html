Реализация приложения (удаленный вызов)

Модификация программного кода классов DemoFunction, DemoAdapter, DemoApplication и JSP для реализации удаленного вызова методов функций не требуется.

Изменения вносятся только в конфигурационный файл приложения и контекстный файл web.xml. В файл web.xml изменения необходимы только при использовании протокола HTTP.

Для сохранения работоспособности функции приложения, демонстрирующие локальный вызов функций, созданы копии страницы first.jsp с именами rmi.jsp и http.jsp для демонстрации удаленных вызовов функций с использованием различных протоколов. В этих копиях файлах изменено только имя используемого экземпляра клаcса DemoApplication c DemoApp на RMIApp и HTTPApp соответственно.

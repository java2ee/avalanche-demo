Реализация JSP

Код страницы JSP, который приведен ниже, демонстрирует получение информации об операционной системе. О какой именно операционной системе выводится информация, локальной или любой удаленной, зависит только от конфигурации приложения. 

<%@ page import="ru.grw.demo.avalanche.DemoApplication"%>
<%@ page language="java" contentType="text/html;charset=UTF-8"
    pageEncoding="UTF-8"%>
<!DOCTYPE html PUBLIC "-//W3C//DTD HTML 4.01 Transitional//EN" "http://www.w3.org/TR/html4/loose.dtd">
<html>
<head>
<meta http-equiv="Content-Type" content="text/html; charset=UTF-8">
<title>Демонстрационное приложение</title>
</head>
<body>
<h1>Демонстрационное приложение</h1>
<jsp:useBean id='DemoApp' scope='application' class='ru.grw.demo.avalanche.DemoApplication'/>

<% = DemoApp.getInfo(true) %>

</body>
</html>

Для успешного выполнения кода этой страницы необходимо добавить в контекстный файл web.xml следующие секции <servlet> и <servlet-mapping>. и определик конфигурационный файл приложения.

Реализация JSP-страницы (протокол HTTP)

В модификации кода страницы JSP так же нет какой либо необходимости. Для сохранения работоспособности ранее реализованной JSP страница, сделана ее другая копия, в которой необходимо изменить имя используемого экземпляра класса приложения (Application) c DemoApp на HTTPApp. 

<%@ page import="ru.grw.demo.avalanche.DemoApplication"%>
<%@ page language="java" contentType="text/html;charset=UTF-8"
    pageEncoding="UTF-8"%>
<!DOCTYPE html PUBLIC "-//W3C//DTD HTML 4.01 Transitional//EN" "http://www.w3.org/TR/html4/loose.dtd">
<html>
<head>
<meta http-equiv="Content-Type" content="text/html; charset=UTF-8">
<title>Демонстрационное приложение - HTTP</title>
</head>
<body>
<h1>Демонстрационное приложение - HTTP</h1>
<jsp:useBean id='HTTPApp' scope='application' class='ru.grw.demo.avalanche.DemoApplication'/>

<% = HTTPApp.getInfo(true) %>

</body>
</html>

Экземпляр класса Application с именем HTTPApp использует адаптер, сконфигурированный на вызов удаленной функции по протоколу HTTP.
